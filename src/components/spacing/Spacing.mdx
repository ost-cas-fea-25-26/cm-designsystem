# Spacing

## CSS Variables

<ul>
  <li>--spacing = 4px</li>
</ul>

## Sizes

Sizes are in 4px increments:

<ul>
  <li>spacing-1 = 4px</li>
  <li>spacing-2 = 8px</li>
  <li>spacing-3 = 12px</li>
  <li>spacing-4 = 16px</li>
  <li>spacing-5 = 20px</li>
  <li>spacing-6 = 24px</li>
  <li>spacing-7 = 28px</li>
  <li>spacing-8 = 32px</li>
  <li>...</li>
</ul>

## Usage examples

Insert number of spacing in `<spacing>`

<ul>
 <li>**Padding**: p-[`<spacing>`] => p-2</li>
 <li>**Margin**: m-[`<spacing>`] => m-1</li>
 <li>**Width**: w-[`<spacing>`] => w-10</li>
 <li>**Height**: h-[`<spacing>`] => h-10</li>
</ul>

[See TailwindCss Documentation](https://tailwindcss.com/docs/styling-with-utility-classes)
