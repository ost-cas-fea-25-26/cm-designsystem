# Mumble Branding

Mumble branding assets include logos, app icons, and brand marks in multiple formats and color variations.

## Usage Guidelines

- Use **inline logos** for horizontal spaces (headers, navigation)
- Use **stacked logos** for square or narrow spaces (mobile, social media)
- Use **app icons** for favicons and app launchers
- **Gradient** is preferred for primary branding
- **Violet** works on light backgrounds
- **White** requires dark backgrounds (violet-600 or darker)
